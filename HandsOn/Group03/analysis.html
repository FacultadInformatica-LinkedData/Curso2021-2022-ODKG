Analysis of Data

id

Type: Integer (7 digits)

Description: Unique number to identify each row in the dataset, in this case recycle bins.

tipoResiduo

Type: String

Range: [Resto, Envases]

Description: Attribute that indicates the type of residue which can be stored in each recycle bin. "Envases" means only plastic waste "Resto" menas any other kind of waste.

direccion

Type: String

Description: It indicates the closest address of the recycle bin.

longitud

Type: Float

Description: Number that indicates the exact spatial longitude of the recycle bin.

longitud

Type: Float

Description: Number that indicates the exact spatial latitude of the recycle bin.

nivelLlenado

type: Integer

Range: [0-10]

Description: It indicates the fill level of the recycle bin, from 0 to 10. The higher the number, the fuller it is.

estado

type: String

Range: [OK, LLENA, ERROR]

Description: It indicates the actual state of the bin. "OK" means that the bin is operative, "LLENA" means that the recycle bin is full, and "ERROR" means that there was an error on the lecture of the bin (the values might not be correct).

fecha

Type:DateTime(YYYY-MM-DDTHH:MM:SS)

Description: It told us the last time the information about the recicle bin was collected
