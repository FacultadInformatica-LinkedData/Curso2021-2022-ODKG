Application Requirements Group 03

Use case 01: Get information about a trash bin

The application must be able to allow the user to click on a trash bin to get information about the capacity, the address, id, resource type and last time checked

Use case 02: Filter by resource type

The application must be able to allow the user to change between resource types bins in the map.

Use case 03: Filter by capacity

The application must be able to allow the user to select a certain capacity bins in the map.

Use case 04: Filter by id

The application must be able to allow the user to select the bin that matches with the id in the map.

Requirement 01:

The application has to show the trash bins with a dot in a map

Requirement 02:

The dots that represent the trash bins are of different colors, depending of the type of residue, green if it is leftovers and yellow if it is packages

Requirement 03:

Instead of dots, the trash bins that are full or damaged will be represented by a cross

Requirement 04:

The application will open a pop up when a trash bin is selected

Requirement 05:

The pop up shows:

  • The capacity of the tras bin
  • When was the last time the information about the trash bin was updated
  • Which kind of waste should be deposited in the selected bin
  • The id of the selected bin
  • The address of the selected bin

Requirement 06:

The application will let you search for available bins in the vicinity of a selected location, filtering by residue type, and level of fullness
