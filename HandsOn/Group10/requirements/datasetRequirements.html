R1. Data are in the smart cities domain.

The data sets are in the domain of smart cities since it allows easy access to cultural events in the city of Madrid filtering by dates.

R2. Data are available as a CSV file.

All datasets ara available in CSV

R3. Data have an open license so it can be published.

The Madrid City Open Data Portal provides access to municipal government data and is dedicated to promoting the development of creative tools to attract and serve the citizens of Madrid.

It allows the reuse of documents and data for commercial and non-commercial purposes. Authorized reuse includes activities such as copying, disseminating, modifying, adapting, extracting, rearranging, and combining information. For more information, go here.

R4. Data can be easily link

The data can be easily link with generic real-world entities. For example, the used datasets contain dates of the events, postal adress and localization coordinates.

R5. Documentation exists for the data and Data can come from multiple data sources

All datasets have documentation and it is available here. The datasets have the same structure and documentation because they come from the same source.
