Analyses performed over the data source:

There are a total of 790 instances and 13 attributes described as is mention in the following lines:

1) EOI_Code: String code that identifies the measure station.

2) Station_Name: String referred to the name of the measure station.

3) Date: String that represents the date in the following format: YYYY-MM-DD

4) Pollutant_Code: Numeric code that identifies the contaminant. String values from 9 to 63.

5) Pollutant_Name: Contaminant chemical name (benzene or cadmium).

6) Units: Unit of measure.

7) Station_Type: Type of measure station. The possible strings are background, industrial or traffic.

8) Station_Environment: Environment within the measure station is placed. The possible strings are urban, peri-urban or rural.

9) INE_Code: String code that identifies the municipality where is situated the measure station.

10) Municipality: Municipality where the measure station is located.

11) Value:: Numeric double value of the measure for each instance.

12) Latitude: Numeric double latitude of the measure station.

13) Longitude: Numeric double length of the measure station.

14) Georeference: Point of a geographic map composed by the latitude and length where the measure station is placed.
