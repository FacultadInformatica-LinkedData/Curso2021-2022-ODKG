Analyses performed over the data source:

There are a total of 32247 instances and 45 attributes described as is mention in the following lines:

1) EOI_Code: String code that identifies the measure station.

2) Station_Name: String referred to the name of the measure station.

3) Date: Year (String value from 2010 to 2021), Month (String values from 1 to 12) and Day (String values from 1 to 31) of the measure.

4) Pollutant_Code: Numeric code that identifies the contaminant. String values from 9 to 63.

5) Pollutant_Name: Contaminant chemical formula.

6) Units: Unit of measure.

7) Station_Type: Type of measure station. The possible values are background, industrial or traffic.

8) Station_Environment: Environment of measure station. The possible values are urban, peri-urban or rural.

9) INE_Code: String code that identifies the municipality where is situated the measure station.

10) Municipality: Municipality where the measure station is located.

11-41) Value:: Numeric value of the measure for each day of the month.

42) Altitude: Numeric altitude of the measure station.

43) Latitude: Numeric latitude of the measure station.

44) Longitude: Numeric length of the measure station.

45) Georeference: Point of a geographic map composed by the latitude and length where the measure station is placed.
