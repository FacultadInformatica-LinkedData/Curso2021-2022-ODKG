Dataset requirements

R1: Data are in one of the selected smart city domains. Specifically in the one about Smart environmental protection.

R2: Data are available as a CSV file in datos.gob.es. The dataset is also available in RDF-XML and JSON

R3: Data have an open license, specifically Open Data Commons Attribution License; so they can be published.

R4: Data can be easily linked with generic real-world entities like districts, locations or environmental facilities.

R5: Documentation exists for the data and it is available at datos.gob.es.

R6: All relevant data is available in one source, but the project is easily scalable to incorporate more geographical areas, which would requiere different sources.
