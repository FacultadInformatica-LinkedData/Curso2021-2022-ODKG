Dataset requirements

The selected dataset must fulfill some requirements in order to be acceptable to do the coming assignments. These requirements and the comments made about them (based on the selected dataset) are listed in the following table:

NÂº Description Comment
R1 Data are in one of the selected smart city domains. Data provides the information regarding the location of electrical scooters for renting which is related to the domain Smart Traffic.
R2 Data are available as a CSV file. The data is stored in the file E-Scooter_Trips_2019 which is a csv file located on the csv folder.
R3 Data have an open license so they can be published. As stated in the description of the dataset in this link. The data has a See Terms of Use clause that leads to the following description (More info), that basically states that the City of Chicago does not take care of any issues regarding any error that is on the data and also that in the case that the data is used on an application it should state the following paragraph.
“This site provides applications using data that has been modified for use from its original source, www.cityofchicago.org, the official website of the City of Chicago. The City of Chicago makes no claims as to the content, accuracy, timeliness, or completeness of any of the data provided at this site. The data provided at this site is subject to change at any time. It is understood that the data provided at this site is being used at one's own risk”
R4 Data can be easily linked with generic real-world entities (e.g., locations). This dataset has the attributes Start centroid Latitude, Start centroid Longitude, End centroid Latitude and End centroid Longitude, of the scooters, that way it is possible to link this information to a real world map.
R5 Documentation exists for the data (optional). In the following link there is the documentation and explanation of the schema of data here.
R6 Data can come from multiple data sources (optional). This dataset provides information collected just from the server where scooters are connected to, so it is not multiple sources.
