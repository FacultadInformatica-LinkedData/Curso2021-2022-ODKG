Dataset analysis

There are 2290 records in our dataset in summary. The reports are from 2019 Januar 1st to 2021 June 30th.

Column Type Comments/Range Problems
num_expediente String The identification number of the accident. Not unique (there might be different people in the same accident) Duplicated values because of having more people in the same accident
fecha String Date of the accident, DD/MM/YYYY format. No missing values.
hora String Exact time of the accident, HH:MM format. No missing values. 458 different values.
localizacion String The location where the accident happened. No missing values. There are some ","s in between the values which needs to be corrected. This attribut does not have a well defined format.
numero String The number of the street where the accident happend. No missing values.
distrito String The distinct of the accident. 21 different values (no missing values).
tipo_accidente String The type of the accident. 12 different values (no missing values).
estado_meteorologico String The whether status during the accident. 5 different options, 63 (~2,75%) missing values
tipo_vehiculo String The type of the vehicle. 2 different values (no missing values).
tipo_persona String The type of the person (conductor, pasajero, peaton). 3 different values (no missing values).
rango_edad String The range of the person's age. 18 different values (no missing values). There no missing values, however, one of hte values is "Deconocido". This needs to be handled later.
sexo String The gender of the person. 3 different values. There no missing values, however, one of hte values is "Deconocido". This needs to be handled later.
lesividad Number The harmfulness of the accident. It has a value between 1-15 based on how harmful the accident was. 269 (~11,7%) missing values.
coordenada_x_utm Number The X coordinate of the accident (no missing values). The format of the values is not valid.
coordenada_y_utm Number The Y coordinate of the accident (no missing values). The format of the values is not valid.
positiva_alcohol String The result of the alcohol test of the person. "N" stands for the negative alcohol test, "S" for positive alcohol test. 4 missing values (~0,17%). The missing values need to be handled.
positiva_droga String The result of the drog test of the person. All values are missing (100%). All the values are missing so this column needs to be removed.

Resource naming strategy

Hash URIs for ontological terms, slash URIs for individuals.

  • Domain: http://bicycleaccidents.org/
  • Ontologycal terms: http://bicycleaccidents.org/ontology#
  • Individuals path: http://bicycleaccidents.org/resource/
  • Ontological terms example: http://bicycleaccidents.org/ontology#hasAge
  • Individuals example: http://bicycleaccidents.org/resource/2019S000036
